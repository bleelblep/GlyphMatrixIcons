## NP3 Essential Notification Icons

A collection of ** custom icons designed for NP3 Essential notifications**

This repo contains icon assets made to work well with the **Nothing aesthetic** - clean, minimal, high-contrast, and readable at small sizes.

## About

These icons are intended for use in **Essential Notifications** on **Nothing Phone (3)** 

The goal is to create icons that:

- look good on the glyph matrix
- stay readable at small sizes
- match the clean monochrome / geometric Nothing design languagd
- work well for app, contact, or custom notification use cases

A collection of **PNG icons for NP3 Essential Notifications**.

This repository stores custom icons that can be used with **Nothing Phone (3)** Essential Notifications.

---

## Folder Structure

```text
/apps
/icons
```

- `/apps` → icons for specific apps or services  
- `/icons` → general-purpose or custom icons  

### Example

```text
/apps
  spotify.png
  youtube.png
  gmail.png

/icons
  stop-sign.png
  finger-heart.png
```
